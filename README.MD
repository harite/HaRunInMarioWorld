## ImpactJS 介绍
ImpactJS是一个基于JavaScript的HTML5游戏引擎，同时支持PC和移动平台浏览器。它是目前除了Construct2之外最受欢迎的HTML5游戏引擎，使用需要支付99美元。

优点：
- 提供了灵活的关卡编辑器，可以快速构建游戏地图
- 提供了强大的调试工具
- 提供了Ejecta可以将JavaScript的执行结果通过OpenGL渲染出来，可以在iOS平台上获得与原生应用相近的效率
- 文档齐全，有两本专门介绍ImpactJS开发的书
- 支持物理效果
- 支持自己编写插件来扩展
